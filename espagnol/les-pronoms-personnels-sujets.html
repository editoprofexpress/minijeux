Pronoms sujets en espagnol
En espagnol, le pronom sujet est souvent omis car la terminaison du verbe suffit. Mais il faut savoir les reconnaître et les utiliser pour insister.
PronomTraductionExemple
yojeYo hablo español.
tútuTú eres simpático.
él / ella / ustedil / elle / vous (formel)Él trabaja mucho.
nosotros / nosotrasnousNosotros vivimos aquí.
vosotros / vosotrasvousVosotros coméis bien.
ellos / ellas / ustedesils / elles / vous (formel pl.)Ellas estudian mucho.
