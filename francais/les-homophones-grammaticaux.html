Homophones grammaticaux
Choisis le bon homophone dans chaque phrase. Chaque erreur est expliquée pour t'aider à progresser !
Rappels clés :
→ a / à : "a" = verbe avoir (remplace par "avait") · "à" = préposition
→ on / ont : "on" = pronom (remplace par "il") · "ont" = verbe avoir
→ son / sont : "son" = déterminant (remplace par "mon") · "sont" = verbe être
→ ces / ses / c'est / s'est : déterminants vs verbes
→ ou / où : "ou" = choix · "où" = lieu/temps
