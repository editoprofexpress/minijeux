Complète les termes suivants
Une suite est définie par $u_{n+1} = a \cdot u_n + b$. Applique la formule pour trouver le terme demandé.
Méthode :
Pour calculer $u_{n+1}$, on remplace $u_n$ dans $u_{n+1} = a \cdot u_n + b$.

Point fixe : $L = \dfrac{b}{1-a}$ — si $u_0 = L$, la suite est constante.
Si $|a| < 1$, la suite converge vers $L$.
